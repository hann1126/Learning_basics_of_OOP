## ğŸ” Penjelasan Setiap Folder

### **1. abstraction/**
Berisi contoh bagaimana membuat class abstract dan bagaimana class turunan mengimplementasikan method yang bersifat abstract.

### **2. inheritance/**
Berisi contoh pewarisan (inheritance), termasuk:
- Class induk & class turunan  
- Pewarisan berlapis (multi-level inheritance)  
- README khusus untuk penjelasan mendalam

### **3. interfacee/**
Memuat contoh penggunaan interface:
- Kontrak perilaku antara class  
- Polimorfisme berbasis interface  
- Latihan seperti pembayaran digital dan kendaraan

### **4. polymorphism/**
Contoh polimorfisme:
- Overriding method  
- Dynamic binding saat runtime
